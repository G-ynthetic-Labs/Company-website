# Cubex3 Command Nexus

A powerful React application built with TypeScript, Three.js, and TailwindCSS.

## Features

- **3D Visualization**: Leverages Three.js and React Three Fiber for immersive 3D interfaces.
- **AI Integration**: Built with Google's Generative AI capabilities.
- **Modern UI**: Styled with TailwindCSS for a sleek, responsive design.
- **State Management**: Uses Zustand for efficient state handling.

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/cubex3-command-nexus.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd cubex3-command-nexus
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Application

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
